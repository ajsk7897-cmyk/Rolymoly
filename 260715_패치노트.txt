# 2026-07-15 업데이트 및 수정 내역 요약

### 1. 비대칭 내전 포인트(MMR) 및 즉각 강등 로직 적용
* 비대칭 포인트 증감: 승리 시 상위 티어 격차, 패배 시 하위 티어 격차를 기준으로 점수 변동 (예: 마스터 승리 시 +50점, 패배 시 -14점).
* 칼같은 티어 강등: 파워스코어가 커트라인 밑으로 떨어지면 즉시 하위 티어로 강등되도록 완충 구간 제거.
* 과거 기록 소급 적용: 과거 모든 내전 전적에 위 로직을 적용하여 전체 회원의 포인트 변동폭(match_bonus) 재계산 및 DB 업데이트 완료. (22명 누적 포인트 정상화)

### 2. 운영진 전용 페이지(회원 관리) 편의성 대폭 업그레이드
* 회원 요약 정보 완벽 틀고정: 회원 표 맨 앞에 [티어] 보상기호 닉네임#태그 형태의 단일 요약 열을 고정 인덱스로 생성하여 틀고정 기능 완벽 구현.
* 직관적인 우승 보상(기호) 지급: 🏆트로피, 🎖️메달, ⭐별 개수를 각각 따로 입력받아 자동 합산 및 저장되도록 UI 개편.
* 보상 로직 안내문 추가: 기존 자동 누적 보상과 수동 지급 보상이 합산되는 원리를 명시한 접이식 안내 탭 추가 및 셀렉트 박스 높이 등 레이아웃 정돈.
* 토스트(Toast) 알림 기능 도입: 권한 변경, 강제 탈퇴 등 운영진 조작 시 우측 하단에 잔존하는 시각적 성공/경고 팝업(Toast) 알림 적용.

### 3. 경매 내전 진행 UI 개선
* 경매 대상 텍스트 정돈: 내전 탭에서 대상자 표기 텍스트 크기를 80%로 축소하고 줄바꿈/여백 정리.
* 보상 기호 동시 표기: 대상자의 닉네임과 함께 기존에 획득한 트로피(🏆), 메달(🎖️), 별(⭐) 기호가 즉시 노출되도록 개선.
