# 롤 클랜 관리자 웹 애플리케이션 사용 설명서

본 프로젝트는 Streamlit 기반으로 제작된 롤(League of Legends) 클랜 내전 및 회원 관리 웹 앱입니다.
Google 스프레드시트를 데이터베이스(DB)로 사용하여 별도의 서버 없이 가볍게 운영할 수 있습니다.

앱을 정상적으로 구동하기 위한 초기 세팅 방법을 아래 안내해 드립니다.

---

## 1. 필수 프로그램 설치
1. **Python 설치**: Python 3.9 이상의 버전을 설치해 주세요. (설치 시 "Add Python to PATH" 체크 필수)
2. **패키지 설치**: 폴더 안의 `run.bat` 파일을 더블클릭하여 실행하시거나, 터미널(CMD)을 열고 폴더 경로로 이동한 뒤 아래 명령어를 입력하세요.
   ```bash
   pip install -r requirements.txt
   ```

---

## 2. 구글 클라우드 플랫폼(GCP) Service Account 세팅
앱이 구글 스프레드시트를 조작하려면 "서비스 계정(Service Account)" 권한이 필요합니다.

1. [Google Cloud Console](https://console.cloud.google.com/)에 접속하여 새 프로젝트를 생성합니다.
2. "API 및 서비스" -> "라이브러리"에서 **Google Sheets API**와 **Google Drive API**를 검색하여 각각 '사용 설정(Enable)' 합니다.
3. "API 및 서비스" -> "사용자 인증 정보(Credentials)" 탭으로 이동합니다.
4. 상단 "사용자 인증 정보 만들기" -> "서비스 계정"을 클릭하고 계정을 생성합니다.
5. 생성된 서비스 계정 목록에서 해당 계정 클릭 -> 상단 "키(Keys)" 탭 -> "키 추가" -> "새 키 만들기" -> **JSON 형식**을 선택하여 다운로드합니다.
6. 다운로드된 파일의 이름을 **`credentials.json`** 으로 변경한 뒤, 앱 메인 폴더(`app.py`가 있는 곳) 안에 넣습니다.
7. 다운로드된 파일 내부 혹은 GCP 화면에서 서비스 계정의 이메일 주소(예: `something@project.iam.gserviceaccount.com`)를 복사해둡니다.

## 3. 구글 스프레드시트 초기 세팅
1. 구글 드라이브에서 새 **Google 스프레드시트**를 하나 생성합니다.
2. 스프레드시트 상단 오른쪽의 **[공유]** 버튼을 누르고, 앞서 복사해 둔 "서비스 계정의 이메일 주소"를 추가하여 **편집자(Editor)** 권한을 부여합니다.
3. 시트 탭 이름을 아래와 같이 변경 및 추가하여 **정확히 4개의 탭**을 만들어주세요. (대소문자 일치)
   - `settings`
   - `users`
   - `matches`
   - `match_players`

### 3-1. settings 시트 초기 설정
앱 구동을 위해 `settings` 시트에 아래와 같이 값을 입력합니다.

| A열 (키) | B열 (값) |
|---|---|
| admin_password | 관리자페이지비밀번호(원하는암호입력) |

### 3-2. users 시트 초기 설정
1행(A1~O1)에 아래 컬럼명(Header)을 순서대로 복사해서 넣어주세요.
`id`, `riot_id`, `tag_line`, `birthdate`, `status`, `solo_tier`, `flex_tier`, `power_score`, `manual_score`, `manual_stars`, `is_admin`, `created_at`, `match_bonus`, `main_position`, `sub_position`

### 3-3. matches 시트 초기 설정
1행(A1~E1)에 아래 컬럼명을 순서대로 넣어주세요.
`id`, `match_type`, `host`, `match_date`, `winning_team`

### 3-4. match_players 시트 초기 설정
1행(A1~F1)에 아래 컬럼명을 순서대로 넣어주세요.
`id`, `match_id`, `user_id`, `team_name`, `role`, `points_spent`

---

## 4. config.py 파일 설정
앱과 스프레드시트를 연결하기 위해 ID를 입력해야 합니다.

1. 만든 구글 스프레드시트의 주소창 URL을 확인합니다.
   - 예: `https://docs.google.com/spreadsheets/d/1ABCDEFG_1234567/edit#gid=0`
2. `d/` 와 `/edit` 사이의 긴 문자열(`1ABCDEFG_1234567`)이 스프레드시트 ID입니다. 복사합니다.
3. 앱 폴더 안의 **`config.py`** 파일을 메모장 등 편집기로 엽니다.
4. `SPREADSHEET_ID` 값을 복사한 ID로 변경합니다.
   ```python
   SPREADSHEET_ID = os.getenv("GOOGLE_SHEET_ID", "1ABCDEFG_1234567")
   ```

---

## 5. 앱 실행 및 접속
모든 설정이 완료되었습니다!

1. 폴더 내의 **`run.bat`** 파일을 더블클릭하여 실행합니다. 
   (혹은 터미널에서 `streamlit run app.py` 입력)
2. 까만 도스 창(콘솔)이 뜨면서 로컬 서버가 열리며, 브라우저에 `http://localhost:8501` 이라는 주소로 웹 사이트가 자동으로 팝업됩니다.
3. 좌측 "가입" 메뉴에서 아이디를 등록하고, "관리자 페이지"에 로그인하여 가입자를 승인하는 방식으로 관리를 시작하시면 됩니다!

즐거운 클랜 운영 되시길 바랍니다! 🎉
